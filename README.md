# Indecision

### [watch live](https://indecision-one.herokuapp.com/)

A static page buld under React.js library that let you input a list of tasks to complete.  
Then it pick a task randomly for you

### Requirements 
  * node.js
  * Express

### To run it locally clone then
```javascript
  cd indecision
  yarn install
  yarn run dev-server
```
